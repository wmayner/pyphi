In actual causation, cause-direction background units are now conditioned
on the after-state (time |t|) rather than the before-state (time |t-1|).
For a partial analysis — where the occurrence (cause ∪ effect units) is a
strict subset of the network — the cause repertoire weights candidate past
states using the realized present state that contains the effect being
explained, keeping both halves of the inversion anchored to the same time.
The effect direction continues to condition its background on the
before-state, matching the cause occurrence it explains.

This changes results only for partial analyses. When the occurrence spans
the whole network there are no background units, so every whole-network
example (including the worked examples in Albantakis et al. 2019) is
unaffected.
