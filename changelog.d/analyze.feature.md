Added `pyphi.analyze(substrate, state, *, subset=…, formalism=…, compute=…)`, a high-level entry point that analyzes a single candidate system and returns an `Analysis` bundle exposing the system irreducibility analysis (`.sia`), the cause-effect structure (`.ces`), and the scalar Φ (`.phi`) uniformly across formalisms, with a flat rich-card repr and a one-row `.to_pandas()` summary. `formalism=` switches the formalism for the call (`"IIT_3_0"` / `"IIT_4_0_2023"` / `"IIT_4_0_2026"`); `compute="sia"` or `"ces"` returns the raw result object instead of the bundle, and a callable returns `compute(system)`.
