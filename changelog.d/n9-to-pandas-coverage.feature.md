Extended the labeled-export (`to_pandas()`) convention to every user-facing result type: the system irreducibility analyses (IIT 4.0 and 3.0), the cause-effect structure, the relations family, the actual-causation family (`AcSIA`, `AcRIA`, `CausalLink`, `Account`), `Complex` / `ExcludedCandidate`, the structural types (`System`, `Substrate`, `Node`, `FactoredTPM`, `JointTPM`), partitions/cuts, and `UnitState`. Scalar-record results return a `Series`; collections and structural results return a `DataFrame`. A coverage test now guarantees every displayable result type exports to pandas.
