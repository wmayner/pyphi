Added ergonomic loading helpers to `pyphi.serialize`: path-based `save`/`load` (wire format inferred from the file extension), top-level `pyphi.save`/`pyphi.load`, and `.save()`/`.load()` methods on the user-facing result objects (`result.save("r.json")`, `CauseEffectStructure.load("r.json")`). Removed the redundant `pyphi.substrate.from_json` loader; use `pyphi.load` / `Substrate.load` instead.
