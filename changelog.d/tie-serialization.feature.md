Tied-set metadata round-trips through ``to_json`` / ``from_json`` for :class:`pyphi.models.StateSpecification`, :class:`pyphi.models.RepertoireIrreducibilityAnalysis` (partition ties and state ties), and :class:`pyphi.formalism.iit4.SystemIrreducibilityAnalysis`. Peer entries serialize as shallow dicts under ``_tie_peers`` (or ``_partition_tie_peers`` / ``_state_tie_peers`` for RIA), bypassing the jsonify cache so partition-tied peers — which share the same hash — survive deserialization as distinct objects. After loading, every member of a tied set shares the same tuple.
