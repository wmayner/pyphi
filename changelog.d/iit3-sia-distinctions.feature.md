The IIT 3.0 ``IIT3SystemIrreducibilityAnalysis`` now exposes the unpartitioned cause-effect structure as a ``distinctions`` attribute, alongside the existing ``partitioned_distinctions``. The unpartitioned distinctions are a required input to the big-Φ computation (Φ is the distance between the unpartitioned and partitioned cause-effect structures), so they are already computed during the analysis and are now attached to the result rather than discarded. ``pyphi.formalism.iit3.ces()`` reuses these attached distinctions instead of recomputing them. The attribute is ``None`` only when the analysis short-circuits to zero Φ before the cause-effect structure is computed (e.g. a system that is not strongly connected); in that case the unpartitioned distinctions are still available via ``iit3.ces(system).distinctions``.
