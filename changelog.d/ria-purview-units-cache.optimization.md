``RepertoireIrreducibilityAnalysis.purview_units`` (and ``MICE.purview_units``, which delegates to it) is now a ``cached_property`` instead of a plain property. The same cause/effect MICE participates in many relations and relation faces, each of which rebuilt the frozenset of purview ``UnitState`` objects from scratch; caching builds it once per analysis (hundreds-fold fewer rebuilds on structures with many relations).
