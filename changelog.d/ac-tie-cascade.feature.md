Actual causation tie-resolution is unified through the cascade primitive (Albantakis et al. 2019, Definition 1 + Exclusion). ``Transition.find_mip`` routes through :func:`pyphi.resolve_ties.resolve_ac_partition_tie` (``argmin |alpha|`` with a lex-canonical partition fallback for reproducibility). ``Transition.find_causal_link`` routes through :func:`pyphi.resolve_ties.resolve_ac_causal_link_tie` (``argmax alpha`` → minimality filter dropping strict-superset purviews → lex-smallest representative when symmetric over-determination leaves multiple minimal candidates). :class:`pyphi.models.CausalLink` exposes the tied :class:`pyphi.models.AcRepertoireIrreducibilityAnalysis` set via ``.purview_ties``; ``AcRepertoireIrreducibilityAnalysis`` exposes ``.partition_ties``. Both round-trip through JSON. ``CausalLink.extended_purview`` lists tied minimal purviews as before.
