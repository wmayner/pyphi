``System.effect_marginal`` now returns a ``FactoredTPM`` for all substrates; the binary-only state-by-node (SBN) bridge is retired. ``System.proper_effect_marginal`` likewise returns a ``FactoredTPM`` (one factor per system output unit), mirroring ``proper_cause_marginal``. Subsystem-level state-reachability validation now covers k>2 substrates, which were previously assumed reachable. No change to Φ or AC results.
