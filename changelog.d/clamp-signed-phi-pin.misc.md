Partitions whose raw integration is non-positive map to clamped ``phi = 0`` via the ``|·|+`` operator (Eqs 19-20). At the default ``mip_tie_resolution`` and ``sia_tie_resolution`` chains, ``signed_phi`` is diagnostic-only: clamped-phi ties surface to higher cascade levels and resolve via canonical lex order. The principle of maximal existence treats clamped-zero partitions as not existing at the system level, so a less-negative signed value confers no additional existence.
