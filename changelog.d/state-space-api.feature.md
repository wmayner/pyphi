``FactoredTPM`` and ``Substrate`` constructors now accept ``state_space=`` (a uniform-flat alphabet size or per-node tuple of state labels) instead of ``alphabet_sizes=``. ``Substrate`` also accepts ``alphabet=k`` as a shortcut for a uniform integer alphabet. State-as-labels is resolved to integer indices via the substrate's state_space mapping. ``alphabet_sizes`` is now a derived property on ``FactoredTPM``.
