Subsystem internals rewritten as a layered architecture in `pyphi.core`. The new `CandidateSystem` value type (`(CausalModel, state, node_indices, cut)`) replaces the legacy `Subsystem` class as the canonical candidate-system representation. Repertoire computations are now stateless module-level functions in `pyphi.core.repertoire_algebra` with module-level memoization keyed on `id(cs)` and weakref-based cache eviction. The legacy `Subsystem` class is preserved during this transitional release for test compatibility but is no longer used by the formalism or compute layers.
