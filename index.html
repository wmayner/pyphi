PyPhi¶

PyPhi is a Python library for computing integrated information.

To report issues, please use the issue tracker on the GitHub repository. Bug reports and pull requests are welcome.

Getting started¶

The pyphi.network object is the main object on which computations are performed. It represents the network of interest.

The pyphi.subsystem object is the secondary object; it represents a subsystem of a network. \(\Phi\) is defined on subsystems.

The pyphi.compute module is the main entry-point for the library. It contains methods for calculating concepts, constellations, complexes, etc.
